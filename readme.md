# Wiqi

### About 

This project is a Wikipedia Query Interface for Laravel 5. 
It makes it easy to perform the "action=query" param on the Wikipedia API.

### Installation

Composer:
```js
{
    "require": {
        "likey/wiqi": "dev-master"
    }
}
```
